# 变量

<h3>变量声明</h3>


<div class="admonition admonition-danger">

   修饰符  变量名 : 变量类型 = 初始值

   let | var   &nbsp;变量名 [: 数据类型] [= 初始值]

</div>

```javascript
var info: String = "你好，仓颉！"

var r = 80  // 初值明确，可省略类型

var age : Int64

const PI = 3.14   // 常量严格模式，必须初始化

let Name = "Cangjie"   // 不可变量，只允许初始化一次
```



<h3>标识符</h3>

<div class="admonition admonition-danger">
  
  数字、字母、下划线组成  
  不能以数字开头  
  不能使用关键字

</div> 

```cangjie
变量名（snake_case，全小写 + 下划线，见名知意）:

product_id      产品ID

user_age        年龄
total_amount    金额 
order_count     数量

save_order   保存订单

User    用户

StringUtils 字符串处理类
```
